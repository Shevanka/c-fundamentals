# Struct

This folder demonstrates structured data modeling using C struct.

## Topics Covered

- Struct declaration
- Struct initialization
- Array of struct
- Nested struct
- Simple data management system

## Learning Objectives

- Model real-world data using struct
- Organize related data fields
- Build simple data systems using arrays and struct

## Example Compilation

```bash
gcc student_struct_system.c -o student
./student
```
