# Control Flow

This folder contains programs that demonstrate decision-making logic in C using:

- if
- if-else
- nested if
- switch-case

## Learning Objectives

- Understand conditional branching
- Practice logical operators
- Implement basic decision-based programs

## Example Compilation

```bash
gcc even_or_odd.c -o even
./even
```

## Expected Skills

After completing this section, i am able to:
- Build simple decision-based programs
- Handle multiple condition branches
- Use switch-case properly
