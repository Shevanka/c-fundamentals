# Strings

This folder focuses on manual string manipulation in C.

## Topics Covered

- Manual string length calculation
- Manual string copy
- String comparison
- Case-sensitive vs case-insensitive comparison
- Reverse string implementation

## Learning Objectives

- Understand null-terminated strings
- Traverse character arrays safely
- Implement common string operations without relying fully on <string.h>

## Example Compilation

```bash
gcc reverse_string.c -o reverse
./reverse
```
