# Functions

This folder demonstrates modular programming in C.

## Concepts Covered

- Function declaration and definition
- Parameter passing
- Pass by value
- Pass by reference
- Functions with return values
- Functions without return values

## Learning Objectives

- Separate logic from main()
- Improve code readability
- Build reusable functions

## Example Compilation

```bash
gcc pass_by_reference.c -o reference
./reference
```
